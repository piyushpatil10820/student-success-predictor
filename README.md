# 🎓 Student Success Predictor & Advisory Agent

## 📌 Overview
This project is an AI-based system that predicts whether a student is likely to pass or be at academic risk based on simple daily habits. Instead of focusing only on final marks, it considers factors like study hours, attendance, and quiz performance.

Along with prediction, the system also provides basic suggestions to help students improve, making it more practical and useful.


## 🎯 Objective
The main goal of this project is to apply fundamental concepts of Artificial Intelligence and Machine Learning to a real-world problem — helping students understand and improve their academic performance.



## 🧠 Concepts Used
- Supervised Learning (Classification)
- Random Forest Algorithm
- Data Generation and Preprocessing
- Model Training and Evaluation
- Basic Rule-Based Advisory System



## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn



## 📂 Project Structure



## 📊 How It Works
1. A synthetic dataset of around 250 students is generated  
2. Features include study hours, attendance, and quiz scores  
3. A Random Forest model is trained to learn patterns  
4. The model predicts whether a student is likely to pass or fail  
5. Based on the prediction, the system gives simple suggestions  



## 🧪 Example Usage
The program allows users to enter their own data:

- Study hours per day  
- Attendance percentage  
- Quiz score  

Based on this input, it outputs:
- Predicted result (Pass / At Risk)  
- Advice for improvement  



## ⚠️ Limitations
- Uses synthetic (generated) data instead of real-world data  
- Prediction depends on predefined thresholds  
- Limited number of features
